## Summary
OpenResponses is a standard API for AI responses. Pico AI Server is 100%
compatible with the OpenResponses specification.

## Specification and reference
- Overview: https://www.openresponses.org
- Specification: https://www.openresponses.org/specification

## Example
Fetch the OpenResponses specification:
```bash
curl https://www.openresponses.org/specification
```

## Errors
See the OpenResponses specification for errors and error formats:
https://www.openresponses.org/specification

## Edge cases
See the OpenResponses specification for limits and edge cases:
https://www.openresponses.org/specification
